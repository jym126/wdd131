Module 1: Introduction to JavaScript

What is JavaScript:

JavaScript is a high-level, interpreted, object-oriented programming language. It was initially created to enhance interactivity in web browsers, allowing developers to dynamically modify the content of a web page, respond to user events, and communicate with the server without reloading the page. Here are some key features of JavaScript:

Client-Side Programming Language:

used for client-side development to enhance the user experience in web browsers. Interpreted: JavaScript is an interpreted language, meaning the code is executed line by line without needing to be compiled beforehand. This means its source code is not fully translated into machine code before execution but is interpreted and run in real-time.

Object-Oriented:

"Object-oriented" refers to a programming paradigm that uses the concept of "objects" to organize and structure code. In object-oriented programming (OOP), objects are instances of classes, which serve as templates for creating objects. Each object can contain data in the form of attributes and

functions in the form of methods.

13. Promises Exercise

Simulate obtaining data from two sources and log the results sequentially using promises and async/await. 

// Promises Exercise
    function getSource1() {
        return new Promise(resolve => setTimeout(() => resolve('Source 1 Data'), 2000));
    }
    function getSource2() {
        return new Promise(resolve => setTimeout(() => resolve('Source 2 Data'), 1500));
    }
    async function fetchSequentially() {
        const data1 = await getSource1();
        console.log(data1);
        const data2 = await getSource2();
        console.log(data2);
    }
    fetchSequentially();

14. Async Function Exercise

Create an asynchronous function simulating a time-consuming task and handle its result and errors. 

// Async Function Exercise
    function simulateAsyncTask() {
        return new Promise((resolve, reject) => {
            setTimeout(() => {
                const success = true;
                if (success) {
                    resolve('Task completed successfully');
                } else {
                    reject('Task failed');
                }
            }, 2000);
        });
    }
    async function executeTask() {
        try {
            const result = await simulateAsyncTask();
            console.log(result);
        } catch (error) {
            console.error(error);
        }
    }
    executeTask();
