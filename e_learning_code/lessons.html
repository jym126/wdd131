HTML Basics

HTML is the structure of your webpage. It consists of tags like: 
<h1>Hello World</h1>

HTML helps create headings, paragraphs, links, and more!

CSS Basics

CSS adds style to your webpage, like colors, layouts, and fonts. 
body { background-color: lightblue; }

CSS uses selectors to apply styles to HTML elements.

JavaScript Basics

JavaScript makes your website interactive and dynamic. 
console.log('Hello, World!');

With JavaScript, you can handle events, create animations, and much more!
