            }
        </script>

JavaScript Variables

Learn about different types of data.

JavaScript: Variables y Tipos de Datos

        <script>
        // Declare variables
        let name = "Alice";    // String
        let age = 25;          // Number
        let isStudent = true;  // Boolean
        
        // Display the values
        console.log("Name: " + name);
        console.log("Age: " + age);
        console.log("Is student: " + isStudent);
        </script>
